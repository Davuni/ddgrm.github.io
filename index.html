| 유의사항 |

- 해당 쿠폰은 똑똑꾸러밍 자사몰에서만 등록과 사용이 가능합니다.
- 쿠폰은 마이페이지 메뉴의 혜택관리(쿠폰) 탭에서 등록 가능합니다.
- 쿠폰 발급 이후 ‘30일’ 이내에 사용 가능합니다.
- 발급 이후 기간 내 사용되지 않은 쿠폰은 자동 소멸됩니다.
- 쿠폰은 결제 시 결제정보 탭에서 사용 가능합니다.
- 해당 쿠폰은 8,000원 이상 결제 시 사용 가능합니다.(배송비 별도)
- 기존에 보유하고 계신 쿠폰과 중복적용 불가합니다.
- 쿠폰 등록, 적용이 되지 않는 경우 1:1 문의 부탁 드립니다.
